fibo

# Summary

An sample library, cli to calculate fibonacci.

# How to build

This program supports `make` to build programs.

## Makefile

use `make` to build static library and dynamic library.

```bash
make clean
make
```
You can build cli with followings.

```bash
make clean
make program
```

Use `all` to build all.

```bash
make clean
make all
```
